## クイックスタート

### 初めて使用する方

1. [使用マニュアル](./USER_MANUAL.md)の「初期セットアップ」セクションを読む
2. Node.jsをインストール
3. 環境変数を設定
4. サーバーを起動
5. ブラウザで `http://localhost:3000` にアクセス

### テスト方法を知りたい方

1. [テストレポート](./TEST_REPORT.md)の「実際の動作テスト手順」セクションを読む
2. 手順に従ってテストを実施

## サポート

ドキュメントを読んでも問題が解決しない場合は、以下を確認してください:

1. Node.jsのバージョン: `node --version`
2. ブラウザの開発者ツール（F12）のConsoleタブ
3. サーバーログ（ターミナルのエラーメッセージ）
