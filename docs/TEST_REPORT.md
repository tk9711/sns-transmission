## 実際の動作テスト手順

システムを実際に動作させてテストする場合の手順です。

### 前提条件
- Node.jsがインストールされている
- `.env`ファイルが正しく設定されている
- API認証情報が有効である

### テスト手順

#### 1. サーバー起動
```bash
cd /Users/tk/Downloads/SNStransmission
npm install
npm start
```

#### 2. ブラウザアクセス
`http://localhost:3000` にアクセス

#### 3. note記事取得
「note記事を取得」ボタンをクリック → 記事カードが表示される

#### 4. 投稿モーダル表示
記事カードをクリック → モーダルが開き、投稿文が自動生成される

#### 5. 投稿文編集
テキストエリアで編集 → 文字数カウンターが更新される

#### 6. X投稿（API認証情報が必要）
「Xに投稿」ボタンをクリック → 成功メッセージ表示

#### 7. Instagram投稿（API認証情報が必要）
「Instagramに投稿」ボタンをクリック → 成功メッセージ表示
